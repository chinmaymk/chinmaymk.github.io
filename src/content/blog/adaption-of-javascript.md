---
title: "Adaption of JavaScript"
date: 2013-10-01
description: "The four phases every developer goes through when learning JavaScript"
---

It's funny how people react to JavaScript. Some get excited, some get terrified, while some start crying. A JavaScript conqueror has to go through four phases to accept the language:

## 1. Ignorance

Typical sentiments — Who cares about JavaScript? Do people actually code in it? OMG! Real programmers code in Java, so I'm just gonna learn Java.

## 2. Hatred

Then comes a project requirement asking developers to learn JavaScript. Typical sentiments — Why are you making me do this? After coding for a while, developers fall prey to JavaScript gotchas. Typical sentiments — No, I'm never going to touch JavaScript again!

## 3. Acceptance

Finally folks understand that JS isn't going anywhere. Then comes the zen moment in JavaScript. Step back from the computer and think about why JS exists.

## 4. Realization

After the zen moment, comes realization of the beauty and simplicity of JavaScript. You get introduced to awesome frameworks like AngularJS, Backbone, jaydata, and tools like grunt and bower — and JavaScript becomes your beloved language.

And remember, this is just the beginning!
